## Project Description

This project generates an audio mashup by:

1. Downloading multiple YouTube videos of a given singer
2. Converting videos into audio
3. Trimming first N seconds from each audio
4. Merging all clips into one single mashup file

The project includes:

- Program 1: Command Line Python Script  
- Program 2: Web-Based Mashup Generator (Flask)

---

## Requirements

- Python 3.10+
- FFmpeg
- yt-dlp
- pydub
- flask
